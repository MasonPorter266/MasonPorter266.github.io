Standard 1

Understand Web Page Development

Standard 1.1: Use Standards Compliant HTML to Create Basic Web Pages

Standard 1.1 focus is on the use of standard compliant HTML to create basic web pages. I am able to show proficiency within standard 1.1 through my use of comments, the horizontal rule, linking web pages and use of proper file names. These can be demonstrated by the home page of my semester project website.

Standard 1.2: Use Styles to Format Web Pages

Standard 1.2 focus is on the use of styles to format web pages. I can demonstrate proficiency for standard 1.2 with the use of embedded styles, controlling the white space, and use of the font property on my Mad Lib website.

Standard 1.3: Demonstrate an Understanding of Advanced CSS Selectors and Properties

Standard 1.3 focuses on demonstrating an understanding of advanced CSS selectors and properties. I can demonstrate my proficiency with this standard through the use of external style sheets, use of div tag, and stylized navigation which is shown on my semester project website.
